RS-Timecode
===

[![Build Status](https://travis-ci.org/MarcAntoine-Arnaud/rs_timecode.svg?branch=master)](https://travis-ci.org/MarcAntoine-Arnaud/rs_timecode)

Supported formats:  
- [x] SMPTE 12M

- [ ] SMPTE 309M
